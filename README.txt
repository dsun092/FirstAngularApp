This is my first attempt at a full AngularJS Project.

1. First start python server in /app folder and run python -m SimpleHTTPServer.
2. Open Browser to localhost:8000
3. Enjoy!

Accomplishements
	* Implemented backend with simple Firebase commands and simplefirebaselogin
	* Re-factored controller functions into different services
	* Implemented SMS tool using Zapier and Twilo
	* Basic use of bootstrap to decorate the site

Knowledge gained
	* Model View Controller framework
	* $scope
	* Services
	* $route
	* Data binding
	* DI
	* Angular Forms
